# Lab P1-04 – Implementação do Transformer Completo

Este projeto implementa a arquitetura completa do **Transformer Encoder-Decoder**, conforme descrito no artigo científico *Attention Is All You Need*.

O laboratório integra todos os componentes fundamentais do modelo, incluindo o mecanismo de atenção, a rede feed-forward, as conexões residuais com normalização e o fluxo completo entre Encoder e Decoder.

A biblioteca utilizada para as operações foi o **PyTorch**.

- Python 3.x  
- PyTorch
